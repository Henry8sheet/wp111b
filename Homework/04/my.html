第一章：HTML

1.1. 什麼是 HTML

HTML（Hypertext Markup Language）是一種用於創建網頁的標記語言。HTML文件是由HTML標記組成的，這些標記用於描述網頁的結構和內容。網頁上的文本、圖像、視頻、音頻等元素都可以通過HTML標記添加到網頁中。

1.2. HTML 範例

<html>
<body>
    <a href="https://youtube.com">Youtube</a>
</body>
</html>

第二章：CSS

1.1. 什麼是 CSS

CSS（Cascading Style Sheets）是一種用於描述網頁外觀和排版的樣式表語言。CSS通過定義網頁元素的樣式和排版方式，可以實現網頁的美化和布局控制。CSS可以將HTML標記和網頁樣式分離，使得網頁的維護和修改更加方便和高效。CSS中包含了很多樣式屬性和選擇器，可以控制網頁元素的顏色、字體、大小、位置、背景等屬性，以及實現網頁的响应式設計。CSS3是CSS的最新版本，增加了許多新的功能和特性，使得網頁的設計更加靈活和豐富。

1.2. CSS 範例

h1 {
    text-align: center;
}
